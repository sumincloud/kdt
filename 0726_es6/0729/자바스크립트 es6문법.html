자바스크립트 es6문법 소개와 목차, 변수 키워드

자바스크립트는 1995년 12월에 브렌든 에이크 개발자에 의해 개발된 스크립트 프로그래밍 언어이다. 2015년도에 자바스크립트 표준인 es6문법이 발표되면서 현재까지 기준으로 많이 사용되고 있음.

바닐라(순수, 모태) 자바스크립트 = 자바스크립트

프레임워크 또는 라이브러리가 적용되지 않은 '순수한' 자바스크립트를 의미한다.

즉, jquery, angular, vue, react같은 라이브러리/프레임워크를 사용하지 않는 순수한 자바스크립트

vanilla는 '기본적인(basic)', '있는 그대로', '특색없는', '평범한(plain)'이란 뜻을 지님.

타입스크립트(TypeScript)

자바스크립트의 슈퍼셋인 오픈소스 프로그래밍 언어이다. 마이크로소프트에서 개발, 유지하고 있으며 엄격한 문법을 지원한다.

C#의 리드 아키텍트이자 델파이, 터보 파스칼의 창시자인 Anders Hejlsberg가 개발에 참여한다.

클라이언트 사이드와 서버 사이드를 위한 개발에 사용할 수 있다.

타입스크립트는 자바스크립트 엔진을 사용하면서 커다란 애플리케이션을 개발할 수 있게 설계된 언어이다.

타입스크립트에서 자신이 원하는 타입을 정의하고 프로그래밍을 하면 자바스크립트로 컴파일되어 실행할 수 있다.

1. 변수키워드 var, let, const의 이해

var : variable의 약자로서 변수선언시 이름 앞에 선언하는 키워드(초창기 변수키워드)

let : 변수 선언시 let키워드로 선언한 변수는 재선언할 수 없으며, { }기준으로 전역변수와 지역변수로 나누어 진다(변하는 값)

const : 변수 선언시 const로 선언한 변수는 블럭범위의 변수 키워드이고, 변수의 값을 변경할 수 없다.(고정된 값)
