02_화살표함수(arrow function)

화살표 함수란 기존의 함수에서 es6에서 새롭게 나온 함수의 형태이다.

기존함수 작성과 실행

    function 함수명(매개변수1, 매개변수2...){  //1번 방식
      실행할 내용;
    }

    const 변수명 = function(){ //2번 방식
      실행할 내용;
    }

    //함수명으로 호출하여 실행
    함수명();

es6문법에서 새로 추가된 화살표함수 작성과 실행

    const 변수명 =()=>{
      실행할 내용;
    }

- 화살표 함수를 쓰면 구문을 더 짧고 간결하게 작성 가능합니다.

매개변수가 한개인 유형

    함수명 = 매개변수 => {표현식};
    함수명 = (매개변수) => {표현식};

매개변수가 한개일 경우는 굳이 담지 않아도 된다. 괄호()를 안써도 상관없다.

화살표 함수와 this키워드

  • this(이것) - 해당 키워드가 속한 객체를 참조할 때 사용
  • 혼자 써있다면 전역 객체를 의미
  • 함수에서는 전역 객체를 의미
  • 메서드 내부에서는 메서드를 소유한 객체를 의미
  • 이벤트는 이벤트를 받는 객체 의미
