03_자바스크립트 es6_class문법

자바언어에서는 클래스 객체를 만들기 위한 설계도.

객체를 생성하기 위한 필드와 메서드가 정의되어 있다.

클래스는 일종의 함수이지만, function키워드 대신 class키워드를 사용한다.

속성 constructor()메서드 내부에 할당한다.

class는 사실상 함수(function)이기 때문에 함수선언과 변수선언식 둘다 선언이 가능하다.

    문법)
    class 클래스명{
      constructor(){
        this.속성명 = 값;
      }
      메서드명(){

      }
    }

class문법에서는 클래스명 작성시 무조건 '첫글자는 대문자'로 작성한다.

    문법) 클래스 호출(실행)
    const 인스턴스명 = new 클래스명();

부모클래스를 상속 받는 자손클래스

상속 : 부모 클래스의 멤버를 자식 클래스에게 물려주는 것.

부모클래스 : 상위클래스, 슈퍼클래스(super)라고 함.

자손클래스 : 하위클래스, 파생클래스라고 함.

부모와 자손을 나눠서 관리하는 이유는 재사용해서 새로운 클래스(class)를 만들기 때문에 중복되는 코드를 줄여주고 유지보수의 시간을 단축하기 위함이다.
