01_리액트 문서만들기

리액트 문서는 2가지 형태로 제작할 수 있다. 하나는 html문서에 배포용 cdn주소를 삽입하여 만드는 방법 두번째는 nodejs 패키지를 설치하여 프로젝트를 생성후 작성하는 방법이 있다.

1. 리액트 배포할 때 사용하는 플러그인 소스를 활용하여 내용 출력해보기

    //리액트 홈페이지에서 제공
    script crossorigin src="https://unpkg.com/react@18/umd/react.development.js"
    script crossorigin src="https://unpkg.com/react-dom@18/umd/react-dom.development.js"
    script src="https://unpkg.com/babel-standalone@6.15.0/babel.min.js"

2. 리액트 앱개발을 위한 환경 구축

node.js : 자바스크립트로 개발 할 수 있는 서버환경을 구축해줌.

node.js는 공식사이트에서 '안정화'버전을 클릭하여 다운로드 하고 설치하여 사용한다.

node.js장점

  • 크롬 V8 자바스크립트 엔진으로 빌드한 자바스크립트 런타임
  • 웹브라우저 환경이 아닌 곳에서도 자바스크립트를 이용한 연산이 가능
  • 리액트 프로젝트를 준비하기 위해 필요한 webpack, babel 등의 도구들을 실행하는데에 사용
  • 패키지 매니저 도구인 npm(node package manager)이 자동 설치
  • npm(node package manager)으로 수많은 개발자가 만든 패키지(재사용가능한 코드)를 설치하고 관리할 수 있음

babel

자바스크립트 컴파일러이며 최신 자바스크립트 문법이라든지, 리액트가 사용하는 jsx문법을 브라우저가 이해할 수 있도록 자바스크립트 문법으로 변환시켜주는 역할을 한다.
