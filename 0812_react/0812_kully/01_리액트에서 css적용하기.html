01_리액트에서 css적용하기

리액트에서 jsx문서에 css를 적용하는 방법에 대해 학습하기

html문서에서 css적용하기 위한 방법에는 3가지가 있다.

  • inline style
  • 내부스타일 - head안에 style선언하여 작성하는 방법
  • 외부스타일 - head안에 작성이 완료된 css문서를 link태그로 연결하여 적용하는 방법

리액트에서 css적용방법

  1. inline style : 태그안에 style={{color:#f00,background:#0f0}}
  2. 변수로 자바스크립트 객체 처리 : style={객체명} 
            const styling = {
          background:#ff0,
          color:#f00
        }

        h3 style={styling} 변수로 자바스크립트 객체처리하여 css적용
  3. 외부스타일 시트를 import방식으로 처리 - import './App.css';
  4. className을 통해서 서식연결 - 적용하고자 하는 태그 안에서 className='클래스명'으로 적용
    html문서에서는 클래스 이름연결시 class='클래스명' or class="클래스명"으로 작성하지만 리액트에서는 ''(홑따옴표)로 작성해야 오류가 나지않음.
    리액트에서는 class='' 를 className=''으로 작성해야 서식적용 => 컴파일 결과물 파일에는 class로 변환되어 나옴.
  5. css Moudle방식으로 연결 - 컴포넌트에 스타일을 작성하다 보면 중복되는 경우가 있는데, css Moudle이라는 기술을 사용하며, css클래스가 중첩되는 것을 방지할 수 있다. css Moudule을 사용하는 파일은 [파일명.css] 대신 [파일명.moudle.css]로 저장해 주면됨. 
            h3 className={style.title} css moudle방식으로 적용하기 /h3

        style.module.css파일 작성
        .title{
          background:#ff0;
          font-size:30px;
          color:#f00;
        }
