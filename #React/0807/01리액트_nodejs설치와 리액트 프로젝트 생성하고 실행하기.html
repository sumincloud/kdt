01리액트_nodejs설치와 리액트 프로젝트 생성하고 실행하기

  • nodejs 사이트에 방문하여 lts(안정화)버전을 다운로드하여 설치
  • 내컴퓨터 > 시작 > 실행 > cmd(명령프롬프트) 실행
  • 버전확인 node -v, npm -v
  • npx설치(execute npm package binaries) : npm i npx -g
  • 버전확인 npx -v

명령프롬프트에서 사용하는 명령어

  • cd 폴더명: change directory의 약자로서 지정한 폴더로 이동한다.
  • dir : 폴더, 파일 리스트 출력
  • cd\ : 최상위 폴더로 이동
  • cd.. : 이전 상위 폴더로 이동
  • cls : clear로서 화면을 깨끗하게 비워줌.
  • 드라이브명: - 현재드라이브에서 다른 드라이브로 이동

vs code에서 cmd등록하여 리액트 프로젝트 설치하기

  • 상단 메뉴에 '터미널'메뉴 클릭
  • 새터미널 생성되면 오른쪽 메뉴 '+'옆 화살표 클릭하여 '기본 프로필선택'하면 vs code에서 항상 cmd창이 나오게 됨.
  • 드라이브 명을 전환하여 프로젝트를 설치할 드라이브로 변경한다.
  • npx create-react-app 프로젝트명(폴더명)
  • npx create-react-app react_class

리액트 실행

npm run start 또는 npm start

리액트 종료

커맨트창에서 ctrl + C(종료)

리액트 학습시 필요한 플러그인 설치

Reactjs code snippets : 리액트 코드 작성시 단축키로 빠르게 코드를 작성할 수 있도록 도와주며 문법체크해 줌.
